UI Pack Toony PRO 1.1.5
------------------------------------------------------------------

	This package contains 300+ textures, 80 prefabs of cartoon style UIs, 26 sample scripts and 20 live demo scenes, etc.

	GUI Animator for Unity UI is included. It helps you add animations to Unity UI objects. See https://www.ge-team.com/en/products/gui-animator-for-unity-ui/

	Features:

		• 80+ UI prefabs.
		• 300+ psd files (UHD size).
		• 20 demo scenes for Portrait and Landscape orientations.
		• 26 sample C# scripts.
		• 15 Particle that compatible with Unity UI Canvas.
		• 39 sounds.
		• GUI Animator for Unity UI is included.

		• Support all build player platforms.
		
	Compatible:

		• Unity 5.4.0 or higher.

	Product pages:
	
		Unity Asset Store: https://www.assetstore.unity3d.com/en/#!/content/44103
		GE Store:		https://www.ge-team.com/en/products/ui-pack-toony-pro/

	Note:
	
		• Scenes and scripts are for examples, so you can tweak sprites or custom set up UI animations to suit your projects.
		• Learn how to create sprite atlas for Unity UI, see tutorial on https://www.youtube.com/watch?v=gbgIA3pwpHc or http://docs.unity3d.com/Manual/SpritePacker.html.
